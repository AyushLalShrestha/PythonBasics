Python Tuples

\n'b'
\n'b'
\n'b'

\n'b'\n'b'\n'b'
\n'b'

\n'b'
\n'b'

A tuple is a sequence of immutable Python objects. Tuples are sequences, just like lists. The differences between tuples and lists are, the tuples cannot be changed unlike lists and tuples use parentheses, whereas lists use square brackets.

\n'b'

Creating a tuple is as simple as putting different comma-separated values. Optionally you can put these comma-separated values between parentheses also. For example −

\n'b'
\n'b"tup1 = ('physics', 'chemistry', 1997, 2000);\n"b'tup2 = (1, 2, 3, 4, 5 );\n'b'tup3 = "a", "b", "c", "d";\n'b'
\n'b'

The empty tuple is written as two parentheses containing nothing −

\n'b'
\n'b'tup1 = ();\n'b'
\n'b'

To write a tuple containing a single value you have to include a comma, even though there is only one value −

\n'b'
\n'b'tup1 = (50,);\n'b'
\n'b'

Like string indices, tuple indices start at 0, and they can be sliced, concatenated, and so on.

\n'b'

Accessing Values in Tuples:

\n'b'

To access values in tuple, use the square brackets for slicing along with the index or indices to obtain value available at that index. For example −

\n'b'
\n'b'#!/usr/bin/python\n'b'\n'b"tup1 = ('physics', 'chemistry', 1997, 2000);\n"b'tup2 = (1, 2, 3, 4, 5, 6, 7 );\n'b'\n'b'print "tup1[0]: ", tup1[0]\n'b'print "tup2[1:5]: ", tup2[1:5]\n'b'
\n'b'

When the above code is executed, it produces the following result −

\n'b'
\n'b'tup1[0]:  physics\n'b'tup2[1:5]:  [2, 3, 4, 5]\n'b'
\n'b'

Updating Tuples

\n'b'

Tuples are immutable which means you cannot update or change the values of tuple elements. You are able to take portions of existing tuples to create new tuples as the following example demonstrates −

\n'b'
\n'b'#!/usr/bin/python\n'b'\n'b'tup1 = (12, 34.56);\n'b"tup2 = ('abc', 'xyz');\n"b'\n'b'# Following action is not valid for tuples\n'b'# tup1[0] = 100;\n'b'\n'b"# So let's create a new tuple as follows\n"b'tup3 = tup1 + tup2;\n'b'print tup3\n'b'
\n'b'

When the above code is executed, it produces the following result −

\n'b'
\n'b"(12, 34.56, 'abc', 'xyz')\n"b'
\n'b'

Delete Tuple Elements

\n'b'

Removing individual tuple elements is not possible. There is, of course, nothing wrong with putting together another tuple with the undesired elements discarded.

\n'b'

To explicitly remove an entire tuple, just use the del statement. For example:

\n'b'
\n'b'#!/usr/bin/python\n'b'\n'b"tup = ('physics', 'chemistry', 1997, 2000);\n"b'\n'b'print tup\n'b'del tup;\n'b'print "After deleting tup : "\n'b'print tup\n'b'
\n'b'

This produces the following result. Note an exception raised, this is because after del tup tuple does not exist any more −

\n'b'
\n'b"('physics', 'chemistry', 1997, 2000)\n"b'After deleting tup :\n'b'Traceback (most recent call last):\n'b'  File "test.py", line 9, in <module>\n'b'    print tup;\n'b"NameError: name 'tup' is not defined\n"b'
\n'b'

Basic Tuples Operations

\n'b'

Tuples respond to the + and * operators much like strings; they mean concatenation and repetition here too, except that the result is a new tuple, not a string.

\n'b'

In fact, tuples respond to all of the general sequence operations we used on strings in the prior chapter −

\n'b'\n'b'\n'b'\n'b'\n'b'\n'b"\n"b'\n'b'\n'b'
Python ExpressionResults Description
len((1, 2, 3))3Length
(1, 2, 3) + (4, 5, 6)(1, 2, 3, 4, 5, 6)Concatenation
('Hi!',) * 4('Hi!', 'Hi!', 'Hi!', 'Hi!')Repetition
3 in (1, 2, 3)TrueMembership
for x in (1, 2, 3): print x,1 2 3Iteration
\n'b'

Indexing, Slicing, and Matrixes

\n'b'

Because tuples are sequences, indexing and slicing work the same way for tuples as they do for strings. Assuming following input −

\n'b'
\n'b"L = ('spam', 'Spam', 'SPAM!')\n"b'
\n'b'

 

\n'b'\n'b'\n'b'\n'b"\n"b"\n"b"\n"b'
Python ExpressionResults Description
L[2]'SPAM!'Offsets start at zero
L[-2]'Spam'Negative: count from the right
L[1:]['Spam', 'SPAM!']Slicing fetches sections
\n'b'

No Enclosing Delimiters

\n'b'

Any set of multiple objects, comma-separated, written without identifying symbols, i.e., brackets for lists, parentheses for tuples, etc., default to tuples, as indicated in these short examples −

\n'b'
\n'b'#!/usr/bin/python\n'b'\n'b"print 'abc', -4.24e93, 18+6.6j, 'xyz'\n"b'x, y = 1, 2;\n'b'print "Value of x , y : ", x,y\n'b'
\n'b'

When the above code is executed, it produces the following result −

\n'b'
\n'b'abc -4.24e+93 (18+6.6j) xyz\n'b'Value of x , y : 1 2\n'b'
\n'b'

\n'b'

Python includes the following tuple functions −

SNFunction with Description

2

len(tuple)
